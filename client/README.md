# AURIX — React Frontend

The React dashboard frontend for the AURIX AI Cyber Defense platform.

## Setup

```bash
npm install
cp .env.example .env    # Set VITE_API_URL=http://localhost:8000
npm run dev             # Runs on http://localhost:3000
```

## Pages / Views
- **Login** — Authentication (default entry point)
- **Sign Up** — New user registration
- **Dashboard** — Overview, threat stats, live monitoring
- **Threat Analysis** — Detailed threat management
- **Activity Logs** — Security event log viewer
- **AI Insights** — AI chatbot for threat analysis
- **File Scanner** — Upload and scan files
- **Link Scanner** — URL threat scanning
- **Settings** — User preferences
- **Recycle Bin** — Deleted items recovery
- **Admin Panel** — (Admin only) Device agent management
